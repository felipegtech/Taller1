Acerca de este sitio

Movie Reviews es un sitio web creado con el propósito de mostrar y aprender cómo funciona Django, así como practicar habilidades de desarrollo web utilizando tecnologías como HTML, Bootstrap y Python.

Aquí puedes buscar y explorar reseñas de películas populares, ver detalles y navegar por una interfaz sencilla, moderna y funcional.

Este proyecto forma parte de un ejercicio académico.

Autor: Felipe Gómez
